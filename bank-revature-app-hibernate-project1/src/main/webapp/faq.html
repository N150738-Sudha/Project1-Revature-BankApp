FAQ



1. What is BankApp Mobile Banking?

BankApp Mobile Banking is a mobile application that allows you to access your bank account(s) using a mobile phone. You can view account related information, transfer funds, pay bills and recharge your mobile and a lot more using this application.Download the application now to make banking a lot more simpler!

2. Who can use BankApp?

An existing BankApp customer with a savings/current account registered with either Internet banking or having an ATM cum Debit Card along with mobile number registered for SMS Banking can avail the mobile banking facility.
